# Task Manager (C)

Simple task manager built in C using file handling.

## Features
- Add tasks
- List tasks
- Remove tasks
- Data persistence using text files

## Technologies
- C

## How to Run

1. Compile the program:
```
gcc tarefas.c -o tarefas
```

2. Run the program:
```
./tarefas
```
